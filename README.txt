1 Yêu cầu môi trường thiết lập
Yêu cầu Máy có cài đặt NodeJs Có thể cài đặt NodeJs tại: https://nodejs.org/en/
Máy có cài đặt mongoDB, có thể cài đặt tại: https://www.mongodb.com/try/download/compass
2 Hướng dẫn chạy phần mềm
• Bước 1: Mở cửa sổ terminal và trỏ tới thu mục chứa source code (trong trường hợp này là thư mục src)
• Bước 2: Gõ lệnh "npm install"
• Bước 3: Gõ lệnh "npm start"
• Bước 4: Mở trình duyệt và truy cập địa chỉ: http://localhost:3000
